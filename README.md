# 望远显微镜

一个面向摄影照片浏览、分类整理和拼图展示的 Windows 看图工具。

它不是重型修图软件，重点是把**超大图切换、标签管理、复杂搜索、全屏整理和可复用拼图**这些摄影日常工作做顺手。

## 下载

请到 Releases 页面下载最新版：

👉 [点击下载最新版](https://github.com/Gelivabilities/wyxwj-photo/releases/latest)

在 Release 页面的 **Assets** 区域中，下载压缩包：

`wyxwj-photo-v2.3.0.zip`

解压后运行：

`PhotoManager.exe`

## 亮点

1. **摄影照片浏览更快**  
   针对 20-60M 超大相机照片做了预览图、原图和缩略图加载优化，切图时优先保证当前照片响应。

2. **整理照片更顺手**  
   支持收藏、批量选择、批量打标签、快捷键打标签、复杂搜索和多维过滤，适合大量照片筛选整理。

3. **全屏时也能整理照片**  
   全屏看图时可以快速选图、打标签和筛片，不用频繁退出全屏。

4. **拼图布局灵活易用，可保存修改**  
   可以像拆表格一样设计拼图布局，工程可保存和重新打开，之后继续换图、调整和导出。

5. **更贴近摄影师的实际需求**  
   支持 HIF/HEIC 浏览和批量转 JPG，能保留标签、收藏、旋转等管理信息，减少看图、整理、转格式之间来回切换。

## 基本功能

- 支持 JPG、JPEG、PNG、HIF/HEIC 等常见摄影/图片格式浏览。
- 支持实际大小、适配窗口、全屏、鼠标隐藏、上一张/下一张、旋转、翻转。
- 支持收藏、批量选择、复制、剪切、粘贴、删除、重命名，并可在资源管理器中显示。
- 支持显示文件名、详细照片信息、缓存状态、排序、乱序和目录刷新。
- 支持 HIF/HEIC 批量转 JPG，并保留照片管理信息。
- 支持从资源管理器、QQ、微信、WPS、截图和剪贴板拖入或粘贴临时图片。
- 拼图支持新建、打开、保存、另存为、导出高清图。
- 拼图支持自定义画布比例、拆分行列、拖动边界，并可放入不同目录图片。
- 拼图内每个格子支持缩放、旋转、翻转、简单调色和撤销/重做。

## 适合谁

- 使用索尼、富士等相机拍摄大量 JPG / HIF / HEIC 照片的人。
- 需要快速筛片、分类、打标签、搜索照片的人。
- 希望在 Windows 上像手机相册一样平铺浏览照片的人。
- 需要临时拼图、保存拼图布局、之后继续修改的人。

## 说明

本软件目前主要面向个人摄影照片管理场景，不追求替代 Photoshop、Lightroom、ACDSee 等大型专业软件。

如果只是想快速看图、筛片、打标签、转 JPG、做简单拼图，它会比较顺手。
